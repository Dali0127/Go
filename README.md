## 小daemon

### 字符串中，中文字符的个数

```go
// // 字符串中，中文字的个数
func main() {
	s := "hello沙河小王子"
	a := 0
	for _, i := range s {
		// fmt.Println('z')
		if i > 'z' {
			a += 1
		}
	}
	fmt.Println(a)
```
